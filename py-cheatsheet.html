Python for the Java Developer: Interview Cheatsheet

This cheatsheet is designed for an experienced Java developer to quickly grasp Python concepts crucial for a coding interview. It focuses on mapping concepts you already know to Python's syntax and idiomatic practices, highlighting what's crucial for a coding interview.

1. The Big Picture: Philosophical Differences

Java Python
Statically Typed: String name = "SDE"; Dynamically Typed: name = "SDE"
Compiled: javac → .class → java Interpreted: .py script is run directly.
Verbose: Explicit type declarations, semicolons, curly braces. Concise: "Code as pseudo-code." Readability counts.
Brace-delimited: { ... } defines code blocks. Indentation-delimited: Whitespace is syntax! A tab or 4 spaces define a block.
Strict OOP: Everything must be in a class. Multi-paradigm: You can write procedural, functional, or OO code.
Key Takeaway: You will type much less. Be mindful that type errors will happen at runtime, not compile time. The lack of braces and reliance on indentation is the single biggest syntax change to get used to.

7. Essential Standard Library Modules for Interviews

Module Key Classes/Functions Use Case
collections defaultdict A dictionary that provides a default value for a non-existent key. Amazing for counting/grouping.
from collections import defaultdict
counts = defaultdict(int)
counts['a'] += 1 # Works even if 'a' isn't a key yet
collections Counter A dict subclass for counting hashable objects. The ultimate tool for frequency counts.
from collections import Counter
counts = Counter(['a', 'b', 'a', 'c', 'a'])
# -> Counter({'a': 3, 'b': 1, 'c': 1})
collections deque "Double-ended queue". Optimized for fast appends and pops from both ends. Perfect for BFS algorithms.
from collections import deque
q = deque()
q.append(1)
item = q.popleft() # O(1) operation
heapq heappush, heappop Implements a min-heap. A must-know for priority queue problems (e.g., Top K Elements). Corresponds to Java's PriorityQueue.
import heapq
heap = []
heapq.heappush(heap, 5)
heapq.heappush(heap, 2)
smallest = heapq.heappop(heap) # returns 2
math math.inf, math.sqrt, etc. For infinity (float('inf') also works), square roots, and rounding.

8. Example: Putting It All Together

Problem: Find the first non-repeating character in a string.

A Java-style approach in Python (functional but not "Pythonic"):

def find_non_repeating_java_style(s: str) -> str:
    # Like a HashMap
    counts = {}
    for char in s:
        if char in counts:
            counts[char] = counts[char] + 1
        else:
            counts[char] = 1
    
    for char in s:
        if counts[char] == 1:
            return char
            
    return "_"

An idiomatic, "Pythonic" approach:

from collections import Counter

def find_non_repeating_pythonic(s: str) -> str:
    # Counter does the first loop for you automatically
    counts = Counter(s)
    
    # The second loop remains, but the setup is cleaner
    for char in s:
        if counts[char] == 1:
            return char
            
    return "_"
